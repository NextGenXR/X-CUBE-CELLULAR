Purpose

The X-CUBE-CELLULAR BSP Modems (part of the STM32 Cellular software expansion for STM32Cube)

  • Enables connectivity over Cellular Networks.
  • The network access technology depends on the Cellular Modem used, it can be 2G as fallback but mainly LTE CatM or NBIoT.
  • To ease support of new modems the SW architecture has splitted modem in Generic part of AT handler and custom parts (one per module/modem).
  • This Release Note deals about the modem itself and the custom parts only, the Generic part of AT handler is part of the Cellular middleware RN.

Carefully read the information concerning the modem you are using because it could be necessary to update the modem FW.

Update History

Main Changes

This V6.0.0 adds the official support of Murata T1SE module (includes T1SC that include Altair ALT1250 modem) and Sequans GMS01Q.

Sequans module GMS01Q is the same as GM01Q but it has internal MFF2 SIM with Orange network.

Note X-CUBE-CELLULAR V6.x deliveries onwards won’t support UG96 modem (3G) anymore. Indeed it is not low power friendly for Cellular IoT solutions.

This verison support the following modems:

  • Quectel module BG96 with Qualcomm modem (LTE CatM, NBIoT and 2G Fallback)
  • Sequans module GM01Q/GMS01Q with Sequans modem (LTE CatM)
  • Murata T1SC module (Altair 1250 modem) included in T1SE module (STM32 L462 + ST4SIM + T1SC + QSPI + voltage regulators)
