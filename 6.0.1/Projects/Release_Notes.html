Purpose

The X-CUBE-CELLULAR Projects (part of the STM32 Cellular software expansion for STM32Cube)

Provides examples for multiple STM32 boards; Discovery and Evaluation boards.

Multiple modems are supported

Stack IP on STM32 or into modem are supported (selection is done by compilation flag)

Available IDEs are IAR, Keil and STM32CubeIDE

Update History

Main Changes

This V6.0.0 is a complete reworking of the Cellular midleware and brings changes to supported HW setups.

There are 4 projects:

  • L496 for BG96 and GM01Q
  • L4S5 for BG96 and GM01Q
  • STWINKT1 for BG96 and GM01Q
  • L462 included in T1SE

Regarding previous deliveries:

  • UG96 modem (3G) has been removed.
  • L475 has been replaced by L4S5 (IoT node v1 to V1.5)
  • STWINKT1B and B-L462E-CELL1 are now officially supported

Each projects contains files to build example binaries for a given hardware setup.

For each projects there is an example, the same for all hardware setups, that demonstrates Cellular middleware capabilities, it is Cellular App.

Cellular App runs one instance and possibly a 2nd one for Echo service(s), also allow Ping usage.

For L462 project there is an additional example, Cellular IoT App, that displays on the OLED screen the sensors data and network information

Note the BSP is only delivered for L462 board, and does not yet include drivers for the 2 QSPI flash memories. It will be added in next delivery.

L462 BSP is only use in Cellular IoT demonstration, Cellular App does not use BSP.

For users that needs BSP usage examples, on L496 and L4S5 boards, see CubeFW L4 deliveries. Current version is 1.17.0.

For users that needs BSP usage examples, on STWINKT1B boards, see st.com for SW examples.

Note GM01Q covers GM01Q and GMS01Q (It is GM01Q including an iSIM with Orange profile)

Applications supported

  • Cellular App (2 echo, ping and throughput)
  • Cellular IoT App (display sensors data and network information on OLED screen)

Main Changes

This V5.2.0 adds the unofficial support of Murata T1SE module (Altair ALT1250 modem) on Discovery board (B-L462E-CELL1). it also adds STWIN (STEVAL-STWINKT1) board unofficial support

Previous versions supported the following modems:

  • Quectel module UG96 with Intel modem (3G, won’t be supported anymore in next delivery)
  • Quectel module BG96 with Qualcomm modem (LTE CatM, NBIoT and 2G Fallback)
  • Sequans module GM01Q with Sequans modem (LTE CatM)
  • Murata module T1SC (LTE CatM, NBIoT) (Unofficial)

Applications supported

  • Echo Client
  • Ping Client
  • Com Client
  • MQTT Client
  • HTTP Client
  • UI Client
