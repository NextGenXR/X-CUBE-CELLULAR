Purpose

The X-CUBE-CELLULAR BSP L462 (part of the STM32 Cellular software expansion for STM32Cube)

This directory contains the the drivers of peripherals on board to demonstrate the capabilities of the B-L462E-CELL1 Discovery board.

Update History

Main Changes

Completly reworked regarding previous release. Sensors, display and EEPROM are supported.

The 2 QSPIs (internal, inside T1SE and external, on the board aside the T1SE) are not supported in this release.

In the next release BSP examples will be also provided.

Note version tag mentionned in this RN for previous delivery was wrong. It was V1.1.2 and not V3.0.

A Doxygen (.chm) file is available to describe the APIs.

supported peripherals

  • hts221 (humidity & temperature)
  • lps22hh (pressure)
  • lsm303agr (accelerometer and magnetometer)
  • ssd1315 (OLES screen)
  • EEPROM
